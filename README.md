## 📌 Project Overview

A fully responsive, production-grade automotive spare parts catalog and ordering platform built for **Ekanayaka Motor Industries**, a spare parts supplier based in Piliyandala, Colombo, Sri Lanka.

The platform serves vehicle owners, garages, and workshops by providing a searchable catalog of **10,000+ spare parts** for Toyota, Nissan, Honda, Suzuki, Mitsubishi, Mazda, Isuzu, and Daihatsu vehicles — with a direct WhatsApp ordering system.

## ✨ Key Features

- 🔍 **Real-time Multi-Filter Search** — Search by part number, vehicle model, brand, and part type across 10,000+ entries instantly
- 📱 **Fully Mobile Responsive** — Hamburger navigation, bottom-sheet product modals, touch-optimized forms, 2-column product grid
- 💬 **WhatsApp Ordering System** — Pre-filled order and inquiry messages sent directly to the business via WhatsApp Business API
- 🏷️ **Dynamic Product Catalog** — Paginated grid with brand filtering, part type filtering, and live search
- 🔎 **SEO Optimized** — Schema.org structured data (LocalBusiness + AutoPartsStore), Open Graph, Twitter Cards, canonical URLs
- ⚡ **Zero-Backend Architecture** — Fully self-contained single-file deployment, no server dependency
